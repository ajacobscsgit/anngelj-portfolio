# Anngel Jacobs Color Palette

Brand direction: Elegant Tech / Founder Engineer / Community Builder

This palette is warm, minimal, polished, and professional. It keeps the site from feeling cold while still giving it enough structure for an engineering, analytics, and entrepreneurship portfolio.

## Core CSS Tokens

| Token | Hex | Use |
|---|---:|---|
| `--color-bg` | `#f7f4ef` | Main site background. Creates the warm cream foundation. |
| `--color-bg-soft` | `#fffaf3` | Soft ivory sections, nav backgrounds, subtle blocks. |
| `--color-surface` | `#ffffff` | Cards, forms, resume previews, clean content containers. |
| `--color-surface-alt` | `#f1ece3` | Tags, chips, quiet dividers, alternate card backgrounds. |
| `--color-text` | `#171717` | Default body text when high contrast is needed. |
| `--color-muted` | `#6f6a63` | Paragraphs, captions, secondary details. |
| `--color-heading` | `#101010` | Main headings and high-emphasis labels. |
| `--color-primary` | `#1f2a44` | Primary buttons, navigation, strong links, major accents. |
| `--color-primary-dark` | `#111827` | Footer blocks, dark CTA sections, video frames. |
| `--color-accent` | `#b8915f` | Eyebrows, highlights, hover states, small premium details. |
| `--color-accent-soft` | `#eadcc8` | Image frames, soft badges, warm highlight fills. |
| `--color-border` | `#ded6ca` | Card borders, separators, form outlines. |

## Pairing Rules

- Use Founder Navy on white or soft ivory for primary calls to action.
- Use Warm Gold sparingly. It should feel like a signature detail, not decoration confetti.
- Keep most backgrounds cream, ivory, or white.
- Use muted taupe for body copy so the site feels soft but still readable.
- Use border beige instead of harsh gray borders.

## Accessibility Notes

- Primary text should be `#171717` or `#101010` on light backgrounds.
- Buttons should use `#1f2a44` with white text.
- Avoid using `#b8915f` for long body text because it is best as an accent.
