# Accessibility Smoke Checklist

## Keyboard and focus

- [ ] Navigate header links using keyboard only.
- [ ] Ensure visible focus indicator appears on interactive elements.
- [ ] Ensure form inputs and submit buttons are keyboard accessible.

## Semantics and labels

- [ ] Verify one clear H1 per page.
- [ ] Verify form controls have associated labels.
- [ ] Verify meaningful alt text on informative images.

## Contrast and readability

- [ ] Check text contrast on primary sections and cards.
- [ ] Check button text contrast in normal and hover states.
- [ ] Ensure text is readable at 200 percent zoom.

## Motion and behavior

- [ ] Confirm any motion does not block content use.
- [ ] Verify smooth-scroll and animated sections remain usable with keyboard.
